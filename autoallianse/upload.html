Инструкция по загрузке заказа через Excel на сайт autoallianse.com

Для облегчения загрузки большого объема записей нами был разработан функционал загрузки заказа через Excel.
Скачать инструкцию
Скачать шаблон заказа Excel 97-2003

Для проценки списка артикулов Вам необходимо:

  1. Что необходимо для загрузки
    1. Быть зарегистрированным пользователем сайта autoallianse.com
    2. Иметь доступ к нашему ftp-серверу для скачивания прайсов. Ftp-сервер находится по адресу ftp.autoallianse.com Логин и пароль для доступа можно получить у Вашего менеджера.
    3. Файл загрузки должен быть сохранен в формате книги Excel 97-2003 (*.xls) согласно шаблону заказа. Скачать шаблон
    4. Наименования брендов, номера деталей и код поставщика  берутся из прайсов с ftp-сервера в точном совпадении.
  2. Описание формата файла

    3. Информация по заказу вводится в Лист1. (в скобках "{ }" указаны адреса ячеек при настройке стиля ссылок в Excel вида R1C1)


    1. Ячейка C1 {R1C3} - Клиентский номер заказа. Используется для дальнейшего отслеживания этого номера во внутренних системах АвтоАльянс и удобства поиска номера заказа, если клиентом используется собственная кодификация заказов.
    2. Ячейка C2 {R2C3} - Имя клиента, не используется при загрузке, но может быть необходимо для хранения внутри файла примечания - имени клиента.
    3. Ячейка A5 {R5C1} и нижеследующие ячейки - порядковый номер записи, используется для отслеживания номера строки в момент загрузки. Позволяет определить строку загружаемого заказа, содержащую ошибку.1
    4. Ячейка B5 {R5C2} и нижеследующие ячейки - наименование производителя, берется из прайсов, загружаемых с ftp.autoallianse.com
    5. Ячейка C5 {R5C3} и нижеследующие ячейки - артикул детали, берется из прайсов, загружаемых с ftp.autoallianse.com
    6. Ячейка D5 {R5C4} и нижеследующие ячейки – ожидаемая цена, используется для дальнейшей проверки системой на отклонение.
    7. Ячейка E5 {R5C5} и нижеследующие ячейки - заказываемое количество.
    8. Ячейка F5 {R5C6} и нижеследующие ячейки - код поставщика, берется из прайсов, загружаемых с ftp.autoallianse.com
    9. Ячейка G5 {R6C7} и нижеследующие ячейки - признак запрета изменения номера (аналогичен флагу "только этот номер"). Для запрета поставьте в поле непустой символ, например слово "да". Примечание: пустые символы - пробел, табуляция, служебные символы перевода строки.
    10. Ячейка H5 {R6C8} и нижеследующие ячейки - сюда можно вписать свой внутренний код, или комментарий к отдельно взятой позиции, но не длиннее 50 символов.
  3. Условия, отрабатываемые при загрузке
    1. При наличии нескольких строк одинаковых позиций их количество суммируется. (если присутствует внутренний код, суммирование не производится).
    2. При наличии нескольких строк одинаковых позиций, у одной из которых проставлен признак запрета изменения номера (см. п. 2.8.) этот признак распространяется на все строки.
    3. При загрузке файла, просьба убедиться в отсутствии "пустых строк", которые могли остаться при удалении из них информации и очистить их путем удаления самих строк. Для этого выделяем нижнюю строку, нажимаем Ctrl+Shift+стрелка курсора вниз, потом правой кнопкой мыши - удаление строки.
    4. Заказ можно загрузить только в пустую корзину.
  4. Дополнения
    1. Формат файлов прайсов, находящихся на ftp, описан файле README.TXT, находящемся в корневом каталоге ftp.autoallianse.com
    2. При возникновении вопросов по загрузке заказа через Excel Вы всегда можете связаться с Вашим менеджером.
